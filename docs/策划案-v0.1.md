  七、AI 推荐引擎（核心差异化）

  不要查表，靠 prompt 推理。架构：

  输入维度：
  ├── 硬约束：剩余时间（周）、年收入、可支配比例、家庭责任
  ├── 性格画像：MBTI（4 维）+ 星座（12）+ 生肖（12）+ 血型（4）
  ├── 人生背景：行业、所在城市、已去过的地方
  └── 偏好权重：冒险 vs 安稳、独处 vs 社交、自然 vs 人文

  LLM 输出：
  ├── Top 10 愿望清单（每个含：理由、预算、时间窗口、备选）
  ├── 性格契合度评分（为什么适合你）
  └── 资金可行路径（按月攒多少钱、什么时候能去）

  Prompt 策略（举例）：

  ▎ "用户是 ENFJ + 处女座 + 属龙 + A 型血，35 岁，程序员，年入 40 万，剩余 2400 周。基于 ENFJ 的利他倾向 +
  ▎ 处女座的完美主义 + 龙年的冒险欲，推荐 3 个能在 5 年内实现的人生体验，每个附预算分解和性格契合度解释。"

  八、数据模型（简化版）

  User {

  九、UI 设计语言（Vision Pro 风格规范）

  核心三要素：

  1. Glassmorphism（玻璃质感）
    - backdrop-filter: blur(40px) saturate(180%)
    - 半透明白色 + 1px 内发光边
    - 阴影：多层柔和投影，模拟环境光折射
  2. 色彩（避开 AI 通用紫）
    - 主色：深邃太空蓝 #0A1628（背景）
    - 强调：温暖琥珀 #FFB547（人文温度）
    - 辅助：极光绿 #7FFFD4（生命/希望）
    - 拒绝：赛博紫、霓虹粉（廉价感）
  3. 字体
    - 中文：思源宋体（标题）+ 思源黑体（正文）
    - 英文：SF Pro / Inter
    - 数字：等宽，让"剩余 2,847 周"有重量感

  关键画面设计：
  - 首屏：黑色宇宙背景 + 缓慢旋转的地球 + 中央"你还有 X 周"巨型数字玻璃质感
  - 清单页：卡片瀑布流，每张卡片浮起 + 视差
  - 地图页：3D 地球 + 想去的地方发琥珀光点 + 已完成的发极光绿光点
